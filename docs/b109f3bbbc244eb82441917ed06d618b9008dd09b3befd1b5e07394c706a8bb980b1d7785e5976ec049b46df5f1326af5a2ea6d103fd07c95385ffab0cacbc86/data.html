6.2 RMySQL

Der findes flere pakker der gør det muligt at forbinde RStudio me den MySQL database. Her brugere jeg en pakke ved navn RMySQL

install.packages("RMySQL")
library(RMySQL)

For at oprette en forbindelse til en MySQL server skal du bruge:

  • MySQL bruger -
  • Password -
  • Databasenavn -

Jeg har en test MySQL database kørende som du kan forbinde til på denne måde:

MinDataBase = dbConnect(MySQL(), user='User_DataBase', password='Password_DB', dbname='Name_DataBase', host='Your hosting')

For at hente data fra MySQL serveren over til R skal du bruge denne kommando:

customers = dbGetQuery(MinDataBase, "select * from Customers")
